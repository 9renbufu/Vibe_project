# 项目名称

VoiceSketch AI

# 技术栈

Frontend:

* React
* TypeScript
* Canvas API
* Web Speech API

Backend:

* FastAPI
* WebSocket
* Pydantic

AI:

* Claude API

# 功能需求

开发一个纯语音控制的绘图工具。

用户不能使用鼠标和键盘。

所有绘图操作均通过语音完成。

## 功能1

语音实时识别

## 功能2

自然语言绘图

示例：

“画一个红色太阳”

自动转换为绘图指令。

## 功能3

复杂场景绘图

示例：

“画一个海边日落场景”

自动拆解为多个绘图动作。

## 功能4

上下文记忆

示例：

“把太阳移到左边”

系统能够识别之前创建的太阳。

## 功能5

语音纠错

自动修复ASR错误。

## 功能6

创意模式

用户输入：

“帮我设计一个赛博朋克城市”

AI自动规划并绘图。

# 输出要求

生成完整项目：

* FastAPI后端
* React前端
* WebSocket通信
* Canvas绘图
* Scene管理
* Claude调用模块
* README
* Docker部署文件

要求代码可直接运行。
